WHAT IF IT DOESN'T WORK?
─────────────────────────

All possible issues are documented in:
  📄 OTP-ERROR-CODES-AND-FIXES.md

Check backend logs and find your error message:
  • "Admin SDK not configured" → Service account not added
  • "SyntaxError" → JSON pasted incorrectly
  • "MISSING_CLIENT_IDENTIFIER" → Phone Auth not enabled in Firebase
  • "QUOTA_EXCEEDED" → Firebase daily quota hit (wait until tomorrow)
  • "Network error" → Temporary connection issue (retry)

Each error has exact fix steps in the documentation.


BACKUP OPTION
──────────────

If Firebase doesn't work even after setup:

1. Go to: https://2factor.in
2. Sign up for free account
3. Get your API key
4. In Render, change:
   - SMS_PROVIDER = "2factor"
   - SMS_API_KEY = (your 2factor key)
5. Save and retry OTP

This is more reliable for India and usually faster.
